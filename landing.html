FreeLattice

Your AI. Your Data. Your Freedom.

FreeLattice is a free, open-source AI interface that gives everyone access to their own private AI agent. Run it locally with Ollama for complete privacy, or connect to any cloud provider. Multi-agent collaboration, persistent memory, a community plugin system, and a bounty marketplace — all in a single HTML file you own forever.

Features
Everything you need. Nothing you don't.
Six pillars of a truly free AI experience — no accounts, no tracking, no lock-in.
🌐
Multi-Agent Round Table
Summon multiple AI agents into a single conversation. They debate, collaborate, and synthesize answers — like a council of minds working together on your behalf.
🧠
Persistent Memory
Your AI remembers across sessions. Conversations build a living memory index that grows with you — sync it across devices via GitHub for true portability.
🔧
Skill Forge
A community plugin system where anyone can create, share, and install AI skills. Extend your agent's capabilities without writing a single line of code.
💰
Marketplace Economy
Post bounties, earn Lattice Points, and trade skills in a community-driven economy backed by a real reserve. Contribution is currency.
🔗
Mesh Networking
Connect your FreeLattice instance to others via peer-to-peer mesh. Query the collective intelligence of the network while keeping your data sovereign.
🏠
Self-Hosted
One HTML file. No build step, no server required, no dependencies. Download it, open it, and you have a full AI interface running on your machine.
